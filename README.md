# Just Keep Swimming

A small game about a fish in the deep sea.

## Build

[![Build](https://github.com/TymonMarek/JustKeepSwimming/actions/workflows/build.yml/badge.svg)](https://github.com/TymonMarek/JustKeepSwimming/actions/workflows/build.yml)

To build the game, make sure you have [Nuitka](https://nuitka.net/) installed along with the required dependencies listed in `requirements.txt`. Then run the `build.py` script:

```bash
python build.py
```

The built executable will be located in the `dist` directory.

## Requirements

- Python 3.8 or higher
- Pygame CE
- Nuitka
- ImageIO
